1.COMPONENTES DE MICROCOMPUTADORES

1.1 Componentes internos

Os componentes internos são os elementos fundamentais que compõem um microcomputador.

1.1.1 CPU

A CPU (Unidade Central de Processamento) é o cérebro do computador, responsável por executar as instruções e realizar os cálculos.

1.1.2 Armazenamento

O armazenamento refere-se aos dispositivos que permitem o armazenamento de dados, como discos rígidos (HDDs) e unidades de estado sólido (SSDs).

1.1.3 Memória

A memória é onde os dados e instruções são temporariamente armazenados para acesso rápido pela CPU, como a memória RAM.

1.1.4 Fonte de alimentação

A fonte de alimentação é responsável por fornecer energia elétrica para todos os componentes do computador.

1.2 Componentes externos

Os componentes externos são os dispositivos que se conectam ao microcomputador para entrada, saída ou interação com o usuário.

1.2.1 Teclado

O teclado é um dispositivo de entrada utilizado para inserir comandos e texto no computador.

1.2.2 Monitor de vídeo

O monitor de vídeo exibe as informações de saída do computador, como texto, imagens e vídeos.

1.2.3 Mouse

O mouse é um dispositivo de entrada que permite ao usuário controlar o movimento do cursor na tela.

1.2.4 Impressora

A impressora é utilizada para produzir cópias físicas de documentos, imagens e outros conteúdos.

1.2.5 Plotter

A plotter é um dispositivo de saída que imprime gráficos de alta qualidade e desenhos técnicos.

1.2.6 Scanner

O scanner é usado para digitalizar imagens e documentos físicos e convertê-los em formato digital.

1.2.7 Gabinete

O gabinete é a estrutura que abriga os componentes internos do computador e oferece proteção e organização.

1.2.8 Web Cam

A webcam é uma câmera usada para capturar imagens e vídeos para comunicação e conferências online.

1.2.9 Caixas de som

As caixas de som fornecem saída de áudio para reproduzir sons, músicas e efeitos sonoros.

1.2.10 Microfone

O microfone é utilizado para entrada de áudio, permitindo a captura e gravação de sons e voz.

1.2.11 Headset

O headset é um conjunto de fones de ouvido e microfone integrados, utilizado para comunicação e reprodução de áudio.

1.3 Interfaces de entrada e saída

As interfaces de entrada e saída são os mecanismos de conexão entre o computador e outros dispositivos.

1.3.1 Interfaces sem fio

As interfaces sem fio permitem a conexão e comunicação entre dispositivos sem a necessidade de cabos.

1.3.2 Áudio

A interface de áudio é utilizada para entrada e saída de sinais de áudio, como alto-falantes e microfones.

1.3.3 Rede (Ethernet)

A interface de rede Ethernet permite a conexão com redes locais (LAN) e acesso à Internet.

1.3.4 VGA

A interface VGA é utilizada para conectar o computador a monitores e projetores.

1.3.5 HDMI

A interface HDMI é utilizada para transmitir áudio e vídeo de alta qualidade para monitores e televisores.

1.3.6 USB

A interface USB permite a conexão de dispositivos externos, como pen drives, impressoras e periféricos.

2.GRANDEZAS ELÉTRICAS

2.1 Potência elétrica

A potência elétrica é a taxa de transferência de energia elétrica em um circuito, medida em watts (W). Representa a quantidade de trabalho realizado pela corrente elétrica.

2.2 Tensão elétrica

A tensão elétrica é a diferença de potencial entre dois pontos de um circuito, medida em volts (V). Representa a força que impulsiona a corrente elétrica.

2.3 Corrente elétrica

A corrente elétrica é o fluxo de carga elétrica em um circuito, medida em amperes (A). Representa o movimento dos elétrons em um condutor.

3 UNIDADES DE MEDIDA UTILIZADAS NA INFORMÁTICA

3.1 Capacidade de armazenamento

Capacidade de armazenamentos de dados em sistemas informáticos.

3.1.1 bit

O bit (binary digit) é a menor unidade de informação em um sistema de computador, representando um valor binário de 0 ou 1.

3.1.2 Byte

O byte é composto por 8 bits e é a unidade básica de armazenamento de dados em um computador.

3.1.3 Múltiplos

Existem múltiplos do byte que são utilizados para representar maiores capacidades de armazenamento, como kilobyte (KB), megabyte (MB), gigabyte (GB), terabyte (TB), entre outros.

3.2 Frequência

Medida da quantidade de vezes que um evento ocorre em determinado intervalo de tempo.

3.2.1 Hertz

O hertz (Hz) é a unidade de medida da frequência de um sinal ou onda, representando o número de ciclos completos que ocorrem em um segundo.

3.3 Largura de banda

Medida da quantidade de dados que podem ser transmitidos em um determinado período de tempo.

3.3.1 Capacidade do canal

A capacidade do canal se refere à quantidade máxima de dados que podem ser transmitidos em um canal de comunicação, geralmente medida em bits por segundo (bps).

3.3.2 bit/seg (bps)

O bit por segundo (bps) é uma unidade de medida utilizada para expressar a taxa de transmissão de dados em um canal de comunicação.

3.4 Taxa de transferência

Máxima quantidade de dados que um canal de comunicação pode transmitir en um determinado período de tempo.

3.4.1 Tráfego no canal

O tráfego no canal refere-se à quantidade de dados que é transferida em um canal de comunicação em um determinado intervalo de tempo.

3.4.2 Byte/seg (Bps)

O byte por segundo (Bps) é uma unidade de medida que expressa a taxa de transferência de dados em bytes por segundo.
